Networking Refresher part 1

Ian Kamau Gikonyo

Aug 27, 2023 • 3 min read

In this article I will be covering a brief networking refresher to get you going.

What is a computer network? Basically a computer network is a group or system of interconnected computers.

Any network in a single area is called a LAN (Local Area Network) and the LAN then connects to the WAN. The internet is an example of a WAN.

The OSI model

OSI stands for Open Systems Interconnection. It was introduced to standardize networks in a way that allowed multi-vendor systems.

The OSI model consists of 7 layers which are referenced to help understand how networks operate.

The OSI layers are:

7. Application layer

6.Presentation layer

5.Session layer

4.Transport layer

3.Network layer

2.Data link layer

  1. Physical layer

There is a simple acronym that can help you grasp the 7 layers: Please Do Not Throw Sausage Pizza Away. (1 > 7).

Lets go through the different layers:

1.Physical Layer

This layer is responsible for carrying data across physical hardware such as ethernet cables. Some devices include Network Interface Cards and Hubs.

2. Data Link layer

This layer is responsible for adding physical addresses to the data being transmitted. The physical addresses in this case are MAC addresses. (Media Access Control address). The source and destination MAC address are added. Switching also occurs in this layer.

3.Network Layer

This layer handles IP addressing and routing. The source and destination IP addresses are added to the data. (Internet Protocol address). A router is basically a doorway out of your internal network to the external network.

4.Transport Layer

This layer adds the transport protocol to be used, such as Transmission Control Protocol (TCP) and the User Datagram Protocol (UDP). A protocol is basically a set of rules and regulations, therefore the transport layer regulates how the data will be transported. Another thing to note is that the source and destination port numbers are also added.

5.Session Layer

This layer is responsible for establishing and terminating connections between devices. (Start and Stop sessions).

6.Presentation Layer

This layer formats the data in a way the receiving application can understand.(JPEG,WMV). Encryption and decryption can also be done in this layer. Compression and decompression can also occur.

7.Application Layer

This is the layer where the application and the user interact/communicate. Application specific protocols are used such as SMTP (Simple Mail Transfer Protocol), FTP (File Transfer Protocol), TELNET to mention just a few.

I should also mention encapsulation and decapsulation. Very briefly: Encapsulation adds information to a packet as it travels to its destination (from layer 7 to layer 1). Decapsulation reverses the process by removing the info, so a destination device can read the original data (from layer 1 to layer 7).

It is important to note that we do not use the OSI model, we just reference from it. We use the TCP/IP model.

TCP/IP model

The TCP/IP stands for Transmission Control Protocol/Internet Protocol and is a model used to standardize computer networking. The model consists of 5 layers. They are:

5.Application Layer

4.Transport Layer

3.Network Layer

2.Data Link Layer

1.Physical Layer

A simple acronym for this is: Please Do Not Throw Away (1 > 5).

Troubleshooting a network

When faced with a networking issue it is important to know how to fix the issue. One should reference the OSI model when troubleshooting a network. I will go through a basic process to do so and please add any steps I may have missed in the comments.

1.Physical layer

Look at the networking hardware being used, are all cables plugged in, is the Network Interface Card or Wireless Adapter working. Are there any faulty cables? Examine all physical networking hardware in use.

2.Data Link Layer

Is the switch working properly, are the cables plugged in correctly.

3.Network Layer

Is the router on or off, is it faulty. Does it have a proper power connection. Is IP addressing done correctly.

4.Transport Layer

What ports are being used, which transport protocol is in use, is it the appropriate protocol.

5.Session Layer

Was the session established, or was it terminated.

6.Presentation Layer

Is the data formatted in the correct way, is the appropriate form of encryption and decryption used. Was the data compressed and decompressed.

7.Application Layer

Does the application function properly. Is it trustworthy and legitimate.

Computer networking skills are essential in the cybersecurity sector and one should develop his computer networks skills in order to stand out. That is all for now. See you in part 2.
